The "receiver.py" script will open the JPEG stream at http://192.168.4.1:5000/video, then read the data as a raw byte stream

The script will detect the individual JPEG frames, decode it with OpenCV, and display it in a window

On Windows, to specifically use the adapter and not the internal WiFi
    1. Go to Settings > Network and Internet > Advanced Network Settings
    2. Disable WiFi
    3. Enable WiFi 2
    4. Connect to VideoStreamPi
    5. Run receiver.py or go to http://192.168.4.1:5000/video to verify functionality
